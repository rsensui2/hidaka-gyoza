# ひだかや

餃子専門店「ひだかや」のランディングページ（Next.js）。

## 開発

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) を開いてください。

## Docker

```bash
docker compose up --build
```

## デプロイ

Vercel にデプロイ済み（GitHub: [rsensui2/hidaka-gyoza](https://github.com/rsensui2/hidaka-gyoza)）。
